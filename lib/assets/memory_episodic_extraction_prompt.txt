  "memories": [
    {

      "context": "Additional context that might be relevant later",
      "relevance_score": 0.8,
      "tags": ["tag1", "tag2"],

        "topic": "main topic",
        "outcome": "positive/negative/neutral"
      }
    }
  ]
}

1. Capture the essence of what happened

3. Assign appropriate relevance scores (0.0 to 1.0)
